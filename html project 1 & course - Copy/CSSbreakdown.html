Selector

A selector is used to target an HTML element to apply styles to it. It can be an element name, class, or ID.


Property

A property is a style attribute that can be applied to an element, such as color, font-size, or margin.


Value

A value is the specific setting for a property, such as "red" for color or "16px" for font-size.


Declaration

A declaration is a combination of a property and a value, written like this: color: red;.


Declaration block

A declaration block is a group of declarations enclosed in curly braces, like this: { color: red; font-size: 16px; }.


Rule set

A rule set is a combination of a selector and a declaration block, like this: p { color: red; font-size: 16px; }.


Class selector

A class is a way to group elements together to apply the same styles to them. It is defined with a period before the class name, like this: .my-class { color: blue; }.


ID selector

An ID is a unique identifier for an element. It is defined with a hashtag before the ID name, like this: #my-id { color: green; }. Unlike class selectors, ID selectors must be unique, meaning they can only be applied to a single element.


Universal selector

The universal selector is used to select all elements on a page. It is represented by an asterisk (*), like this: * { margin: 0; padding: 0; }.


Exploring CSS deeper

exploring design

CSS allows us to create visually appealing designs for our webpages. We can use properties like background-color, border, and box-shadow to enhance the appearance of elements. We can also make our colors more interesting using gradients, to further enhance our design.


common CSS properties

  • Color: Used to change the color of the specified element. When applied to text based elements, it changes the color of the text.
  • Background-color: Used to change the background color of an element.
  • Background-image: Used to set an image as the background of an element.
  • Font-size: Used to change the size of the text.
  • Font-family: Used to change the font of the text.
  • Font-weight: Used to change the thickness of the text.
  • font-style: Used to specify the style of the text. Common values include normal (default), italic, and oblique.
  • text-align: Used to control the horizontal alignment of text within its containing element. Common values include left (default), center, right, and justify.
  • text-decoration: Used to specify the decoration added to text, such as underline, overline, or line-through.
  • line-height: Used to control the amount of space above and below inline elements. It is commonly used to improve readability.
  • letter-spacing: Used to control the space between individual letters in text.
  • word-spacing: Used to control the space between words in text.
  • margin: Used to create space around elements, outside of any defined borders.
  • padding: Used to create space around an element's content, inside of any defined borders.
  • border: used to create a visible border line around elements.
  • border-radius: Used to round the corners of an element's border. Values can be specified in pixels (px), percentages (%), or other units to create rounded corners.
  • width: Used to set the width of an element. Values can be specified in pixels (px), percentages (%), or other units like em, rem, or viewport units (vw).
  • height: Used to set the height of an element. Values can be specified in pixels (px), percentages (%), or other units like em, rem, or viewport units (vh).
  • box-shadow: Used to add shadow effects around an element's frame. Values include horizontal offset, vertical offset, blur radius, spread radius, and color (e.g., box-shadow: 2px 2px 4px rgba(0,0,0,0.3)).
  • display: Used to control how an element is displayed. Common values include block, inline, inline-block, flex, grid, and none (hides the element).
  • position: Used to control the positioning of an element. Common values include static (default), relative, absolute, and fixed.
  • top, right, bottom, left: Used to control the position of an element when it is positioned (e.g., relative, absolute, fixed).
  • overflow: Used to control what happens when content overflows an element's box. Common values include visible (default), hidden, scroll, and auto.
  • opacity: Used to control the transparency of an element. Values range from 0 (completely transparent) to 1 (completely opaque).
  • z-index: Used to control the stacking order of positioned elements. Higher values are stacked on top of lower values.
  • cursor: Used to control the type of cursor displayed when hovering over an element. Common values include default, pointer (hand), and text.

Responsive design

CSS also allows us to create responsive designs that adapt to different screen sizes. We can use media queries to apply different styles based on the device's width, height, or orientation. This ensures that our webpages look good on both desktop and mobile devices.


Putting selectors into practice

Now that we understand the different types of selectors, let's see how we can use them in our CSS. For example, we can use a class selector to apply styles to all elements with a specific class:

.my-class {
    color: blue;
    font-size: 16px;
}
when we want to use this class in our HTML we would write it like this: <div class="my-class">This is a div with a class.</div>

We can also use an ID selector to apply styles to a specific element:

#my-id {
    color: green;
    font-size: 18px;
}

Finally, we can use the universal selector to apply styles to all elements on the page:

* {
    margin: 0;
    padding: 0;
}

Margin, Padding, and Border

These properties all effect the the area surrounding your elements, as well as the spacing between them.We can define each of their dimensions to set different amounts of space between different elements.
margin:
The space outside an element.
padding:
The space inside an element, between the content and the border.
border:
A line that goes around an element, between the padding and the margin.
border-radius:
Rounds the corners of an element's border. When using the border-radius, margin and padding properties, we set the size for the top, right, bottom, and left. We can define their size in a couple ways. The example below sets the margin and padding to 10px on all sides. 
            P {
                border-radius: 10px;
                Margin: 10px;
                padding: 10px;
            }
We can also define the size for each side independantly by spelling out each side we want, ie; margin-top, padding-left, etc. In the example, you can see the use of shorthand syntax to set values for all four sides at once 
            p {
                margin: 10px 20px 15px 5px; /* top right bottom left */
                padding: 5px 10px; /* top/bottom right/left */
                border-radius: 8px 16px 8px 16px; /* top-left, top-right, bottom-right, bottom-left */
            }
This allows you to quickly set multiple values in a single line.
