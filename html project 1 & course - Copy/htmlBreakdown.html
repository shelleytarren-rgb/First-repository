Everything you need to know about HTML

Breakdown of HTML fundamentals

HTML is a markup language used to create web pages. It consists of various elements that define the structure and content of a webpage.

Document structure

An HTML document starts with a doctype declaration, followed by the <html> tag, which contains the <head> and <body> sections.


Elements

An element is everything from the opening tag to the closing tag, including the content in between.


Tags

Tags are the keywords surrounded by angle brackets.
Opening tag: <p>
Closing tag: </p>
Closing tags are identified by the forward slash at the beginning. There are also self closing tags that stand alone and dont require the slash.

Attributes

Attributes provide additional information about an element. They are always specified in the opening tag.


Example of an attribute: 
            In the example below, the href attribute specifies the URL of the link.
        <a href="index.html" title="Go to homepage">Home</a>

Self-closing tags

Self-closing tags do not have a closing tag. They are used for elements that do not have any content, like the line break tag <br>.


Links

Links are created using the anchor tag <a>. The href attribute specifies the URL of the page the link goes to.


Styling

Styling can be applied to elements using the style attribute, which allows you to change colors, fonts, and other properties.


Value

The value of an attribute is the information assigned to it. For example, in the link above, the value of the href attribute is "index.html" (which is the file name of the homepage).


content

Content is the text or other elements that are placed between the opening and closing tags of an element. Typically anything that would appear on the webpage is considered content.


comment

Comments are used to leave notes in the code that will not be displayed on the page. They are written like this: <!-- This is a comment -->.


doctype

The doctype declaration is the first line of an HTML document and tells the browser what version of HTML is being used. It looks like this: <!DOCTYPE html>.


Exploring HTML deeper

Adding images

Images can be added to a webpage using the <img> tag. The src(stands for source) attribute specifies the URL of the image, and the alt attribute provides alternative text for accessibility. Just like the <br> tag, the <img> tag is an empty or self closing tag. We can also use attributes like width and height to change the size of the image.


            This is an example of an image tag with the alt attribute, which is used to describe the image for accessibility purposes. The example below is taken straight from the homepage of this website.
            <img src="webpagebanner.png" alt="Web Development Banner" width="100%">

Structuring text with tags

We're already familiar with the <br> tag, which adds line breaks to the webpage. There's also the <em> (meaning emphasis) tag, which does have an opening and closing tag. this tag makes text italic. Then there's the <strong> tag which makes text bold.
We can also structure the way the content appears in the webpage with the div(division) tag. With the div tag we can structure and style elements together.

Lists

Lists can be created using the <ul> (unordered list) or <ol> (ordered list) tags. Each item in the list is defined with the <li> tag. Unordered lists use bullet points, while ordered lists use numbers.
