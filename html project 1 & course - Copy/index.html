This webpage was built using HTML, which is a markup language used to create web pages. In order to actually build the page we start with a doctype declaration, which tells the browser what version of HTML we are using. The basic structure of an HTML document includes the <html> tag, which contains the <head> and <body> sections. The <head> section contains metadata about the document, such as the title(which appears in the tab or window of the web browser) and links to stylesheets or scripts, while the <body> section contains the content that is displayed on the page. This is the basic structure of an HTML document and on this website we'll delve deeper into not only HTML but CSS and JavaScript.
