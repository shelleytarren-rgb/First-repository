All you need to know about JavaScript

Breakdown of JavaScript fundamentals

JavaScript is a programming language used to create interactive and dynamic content on web pages. It can be included in HTML files using the <script> tag.

Variable

A variable is a named storage location for data. It can hold different types of values, such as numbers, strings, or objects. We use let, and const to create variables.


Naming variables

When naming variables, it's important to choose descriptive names that convey the purpose of the variable. Variable names can include letters, numbers, and underscores, but they must start with a letter. They must also be unique within their scope. Also, a variable name must be all one word with no spaces, so when we want to make a name with multiple words we use camel case, where we start with a lower case letter and then uppercase for the first letter of all other words.


values

Values are the actual data stored in variables. They can be of different types, such as numbers, strings, booleans, or objects. A variable with a stored value looks like this, const city = chicago;


Data Types

JavaScript has several data types, including:

  • String: Represents a sequence of characters, e.g., "Hello, world!" A string is enclosed in quotes.
  • Number: Represents numeric values, e.g., 42 or 3.14
  • Boolean: Represents a true/false value. Booleans are simply written as true or false.
  • Object: Represents a collection of key-value pairs where each key (called a property) is a string, and each value can be any data type (including other objects or arrays).
  • Array: Represents an ordered list of values. An array is enclosed in square brackets, e.g., [1, 2, 3].

Let and Const differences

Both let and const are used to declare variables in JavaScript, but they have some key differences:

  • Scope: let has block scope, meaning it's only accessible within the block it's defined in. const also has block scope, but its value cannot be reassigned.
  • Reassignment: Variables declared with let can be reassigned new values, while const variables cannot be reassigned once they're set.

Objects

An object is a collection of key-value pairs that represent data and functionality. Objects can contain properties and methods.

properties

Properties are characteristics or attributes of an object. They are defined as key-value pairs within an object. For example, if we have an object representing a person, it might have properties like name, age, and occupation.


Methods

Methods are functions stored as properties inside objects. A method lets an object perform actions or operations. Below is an example of an object where greet is a method of the person object.

        const person = {
            name: "Alice",
            greet: function() {
                return "hello";
            }
        }

console

The console is a built-in debugging tool in web browsers and Node.js that allows developers to log messages, inspect variables, and interact with the JavaScript environment. It can be accessed through the browser's developer tools. We can also run JavaScript code directly in the console using console.log() in order to run particular snippets of code. Whenever we put a variable inside console.log, the console itself will display the value.

Function

A function is a reusable block of code that performs a specific task. It can take parameters and return a value.


Syntax

JavaScript syntax refers to the set of rules that define how JavaScript code is written. It includes the use of semicolons to end statements, curly braces to define blocks of code, and parentheses to group expressions. For example, a simple function syntax looks like this:

        function myFunction() {
            console.log("Hello, world!");
        }

Event

An event is an action that occurs in the browser, such as a click or a key press. JavaScript can respond to events using event listeners.


Exploring JavaScript deeper

Updating let variables

Variables declared with let can be updated or reassigned. Here's an example:

        let count = 0;
        count = 1;
        console.log(count); // Output: 1

Understanding operators

Operators are symbols or keywords that perform operations on values. JavaScript has various types of operators, including:

  • Arithmetic operators: Used for mathematical calculations, e.g., + (addition), - (subtraction), * (multiplication), / (division).
  • Comparison operators: Used to compare values, e.g., == (equal), != (not equal), > (greater than), < (less than).
  • Logical operators: Used to combine or negate boolean values, e.g., && (and), || (or), ! (not).

Using conditions

Conditions allow us to execute different code blocks based on certain criteria. In JavaScript, we can use if statements to create conditional logic. Here's an example:

        let age = 18;
        if (age >= 18) {
            console.log("You are an adult.");
        }
